# ppbb

Multi-agent app generated by MetaAgent.

Pattern mode: graph

Agents: author -> illustrator -> bookbinder -> checker

LLMs per agent:
- author: deepseek-ai/DeepSeek-V4-Flash
- illustrator: deepseek-ai/DeepSeek-V4-Flash
- bookbinder: deepseek-ai/DeepSeek-V4-Flash
- checker: deepseek-ai/DeepSeek-V4-Flash

## Run

    pip install -r requirements.txt
    python agent.py "your task"

API keys live in config.json under each agent's entry.

## Debug

system_prompts.json lists each agent's resolved system prompt (base persona + route tail + skills + tool guidance), matching what build_system() sends at startup.

## GUI (optional)

A PySide6 desktop chat window. The agent also runs headless (python agent.py) without it.

    pip install -r requirements.txt
    python gui.py

## WebSocket server + web UI

    python server.py

Open http://0.0.0.0:8765/ in a browser for the chat web UI, or connect a WebSocket client to ws://0.0.0.0:8765 and send JSON frames:
    {"type": "task", "task": "..."}
The server streams {"type": "trace"} messages and finishes with {"type": "result"}. Settings: config.json -> server.
