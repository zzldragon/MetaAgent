# BadDataAnalyst

Multi-agent app generated by MetaAgent.

Pattern mode: chain

Agents: 执行，有记忆

LLMs per agent:
- 执行，有记忆: deepseek-ai/DeepSeek-V4-Flash

## Run

    pip install -r requirements.txt
    python agent.py "your task"

API keys live in config.json under each agent's entry.

## Debug

system_prompts.json lists each agent's resolved system prompt (base persona + route tail + skills + tool guidance), matching what build_system() sends at startup.

## GUI (optional)

A PySide6 desktop chat window. The agent also runs headless (python agent.py) without it.

    pip install -r requirements.txt
    python gui.py
